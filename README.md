# PADAE
Federated learning (FL) enhances intrusion detection by aggregating client model parameters without requiring access to raw data. However, this characteristic also exposes FL to poisoning attacks, which pose significant threats to the robustness and accuracy of the global model. To address this issue, we propose Poisoning Attack Defense and Accuracy Enhancement (PADAE), a model based on the Kolmogorov–Smirnov (K-S) test for federated intrusion detection. PADAE consists of three modules: (1) Model Quality Validation, which applies K-S statistics to identify and reject malicious models; (2) Model Parameter Distribution Detection, which leverages K-S test p-values to detect abnormal parameter distributions such as Partial Data Tampering (PDT); and (3) Contribution-based Model Aggregation, which adaptively weights client models according to their K-S statistics to enhance the accuracy of the global model. Experiments on UNSW-NB15 and CIC-IDS2017 demonstrate that PADAE effectively defends against label flipping and PDT attacks, while significantly improving the accuracy and robustness of federated intrusion detection compared to existing methods.
